Image Segment Actions

Keyboard short-cuts, available within BigDataViewer window.

Ctrl + Left-click Image segments: Toggle segment selection
Ctrl + Shift + N Image segments: Select none
Ctrl + S Iterate selection visualisation mode:
  • Dim not selected
  • Only show selected
  • Color selected bright yellow
  • Color selected bright yellow and dim not selected
Ctrl + L Label mask display: Shuffle categorical colors
Ctrl + M Label mask display: Toggle show as binary mask
Ctrl + B Label mask display: Toggle show as boundaries
Ctrl + N Image sets: Show next image set
Ctrl + P Image sets: Show previous image set
